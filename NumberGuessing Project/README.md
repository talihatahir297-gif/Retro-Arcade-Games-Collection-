# 🔢 Number Guessing Game

A tactical console-based number puzzle game programmed in **Java**. It challenges players to find a randomly generated secret target while integrating a local data logging setup to keep track of gaming statistics. This project demonstrates Java programming, algorithmic logic, and file handling fundamentals.

---

## 📌 Project Overview

This project is a Java implementation of the classic Number Guessing game. It focuses on logic validation, random bounds generation, score calculating, and data persistence while providing an interactive command-line gameplay experience.

---

## ✨ Features

- Pseudo-random number generation across customizable ranges
- Real-time high/low guess indicators and feedback
- Attempts tracking and score calculation system
- Persistent match statistics logging via Java Streams
- Clean input validation and error sanitation

---

## 🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java File Handling / Streams
